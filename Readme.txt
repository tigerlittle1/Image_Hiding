GUI使用方式:

影藏模式:
在中端機輸入 : python GUI.py --mode hid
跳出視窗後選擇原始影像、機密影像及使用的方法，並且勾選是否要使用平行處理後點選Hidding，如果無法隱藏則會提示說無法隱藏，隱藏時介面中按鈕皆無法調整，隱藏完畢後點選sava可以儲存key和偽裝影像。
隱藏後終端機會顯示是否使用平行處理、隱藏時間以及偽裝影像PSNR

提取模式:
在中端機輸入 : python GUI.py --mode ext
跳出視窗後選擇偽裝影像、key及使用的方法，並且勾選是否要使用平行處理後點選Extracting,如果無法提取則會提示說無法提取，提取時介面中按鈕皆無法調整，提取完畢後點選sava可以儲存機密影像。
提取後終端機會顯示是否使用平行處理、提取時間
